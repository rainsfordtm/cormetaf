# Corpus folder: Camcorp-s

Treebank annotation was developed for the prose texts and for a subcorpus
of the octosyllabic texts in the original Camcorp. The prose text
syntactic annotation was used to generate pseudo-verse and is 
preserved in this folder. It isn't included in the corpus and I don't
envisage adding it in the future since parsing norms and technology
have advanced considerably since 2011.

+ `syntax2`: Treebank annotation in `syntax2` format (requires my
    `treetools` scripts).
