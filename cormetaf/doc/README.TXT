CamCorp 0.71V
=============

Updated version of the camcorp designed to be used in the revised (2022) book
study.

Changes from v0.7:

- "Unstressed" list of monosyllables restricts stressed monosyllables to
	*lexical categories* only. This was the policy when writing ch. 3, in
	particular the AN verse study, and the blacklist is based on 
	/book-corp/tools/blacklist_str_allfnc.txt.
- Updated sp files to new version of the scanner.
